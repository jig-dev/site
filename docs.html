Getting Started

Intro

Jig is a live programming environment that aims at making it easy to compute and monitor business metrics.

First, we should highlight that Jig is a lightweight application. It doesn't need a server to run, it doesn't need a database to store data. It's an app that runs on your computer and that does all the processing in memory.

We like to think of it as a nimble tool to explore the dark corners of your business data.
