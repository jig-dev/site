Productivity first

Fire up JIG and start coding immediately. JIG's live feedback loop makes writing business apps faster than ever before.

UI for non-designers

JIG comes with packaged charts, tables and input components, so that you can focus on your business rather than drowning in html and css.

Flex your data

No-code data tools are simple but stiff. As a full programming environment, JIG can adapt to any special case your business may need.

Fully extensible

JIG is designed with business apps in mind, but you can use it to create just about anything. From games to live music production: the sky is the limit!

Safe versioning

Unlike spreadsheets, JIG apps are saved as text files. You can store them in a version control system to track their changes easily.

Secure

JIG runs on your computer, so it will never send your business data across the internet, making it both secure and network-resilient.

Frequently Asked Questions

What language does JIG support?

JIG apps are written in Javascript. Under the hood, JIG runs the Observable runtime. So anything you can do in Observable, you can do in JIG.

What features are not in the beta, but that I can expect soon?

  • Database connectivity. Right now, you can only access REST apis, and local files. Support for MySql, PostGreSQL and others is coming soon.
  • Pre-packaged UI components: user inputs / charts / tables
  • Printing / PDF export.

Can we host JIG apps on a server?

Not with the beta version, but we are working on it.

Is JIG an Electron app?

Yes, JIG's beta runs as an Electron app.
